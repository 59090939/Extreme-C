## How to build?

Run the following commands in order to build the sources of all chapters.

```shell
$ git clone https://github.com/PacktPublishing/Extreme-C
$ cd Extreme-C
$ git submodule init
$ git submodule update
$ mkdir build
$ cd build
$ cmake ..
$ make
```
